# 📺 Streaming Platform Analytics Dashboard

An interactive **Power BI dashboard** designed to analyze and compare content across leading OTT streaming platforms. This project provides insights into movies and TV shows available on Netflix, Amazon Prime Video, and Disney+ Hotstar using interactive visualizations, KPI cards, and business intelligence techniques.

---

## 📌 Project Overview

The dashboard enables users to explore streaming platform data through dynamic filters and interactive reports. It helps identify trends in content distribution, genres, release years, ratings, countries, and platform-specific libraries.

This project demonstrates practical Business Intelligence skills using **Power BI**, including data modeling, DAX measures, interactive slicers, drill-down analysis, and dashboard design.

---

## 🎯 Objectives

- Analyze OTT platform content libraries.
- Compare Movies vs TV Shows across platforms.
- Identify the most popular genres.
- Analyze content distribution by country.
- Understand release year trends.
- Provide executive-level business insights through interactive dashboards.

## 📈 Key Insights

- Netflix contains the largest content library among the analyzed platforms.
- Drama and Comedy are the most frequently available genres.
- The United States and India contribute the highest number of titles.
- Movies significantly outnumber TV Shows across the platforms.
- Content production has accelerated considerably over the last decade.
- Interactive filtering enables deeper exploration by platform, genre, country, and release year.
